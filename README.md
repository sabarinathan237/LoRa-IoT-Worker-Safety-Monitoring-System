
# LoRa and IoT-Enabled Industrial Worker Safety Monitoring and Rescue System

## 📌 Abstract
This project focuses on enhancing the safety of industrial workers operating in hazardous and remote environments. The system integrates LoRa communication with IoT technology to provide real-time health monitoring, location tracking, and emergency alerting.

## 🔧 Components Used
- Microcontroller (ESP8266 / MCU)
- MAX30102 (Heart Rate & SpO₂ Sensor)
- GPS Module
- LoRa Transceiver (SX1278)
- SOS Emergency Button
- Buzzer & LED Indicators

## 📡 Features
- Long-range communication using LoRa
- Real-time health monitoring (Heart Rate & SpO₂)
- Live GPS tracking
- SOS emergency alert system
- IoT dashboard monitoring using Blynk
- Instant alerts to safety personnel

## 🛠️ System Architecture
### 🔹 Transmitter (Worker Unit)
- Collects health data using MAX30102
- Tracks location using GPS
- Sends data via LoRa
- Emergency alert via SOS button

### 🔹 Receiver (Monitoring Station)
- Receives LoRa data
- ESP8266 sends data to cloud (Blynk)
- Displays data on mobile/web dashboard
- Triggers alarm (buzzer) in emergency

## 🌐 IoT Platform
